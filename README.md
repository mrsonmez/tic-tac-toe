## About

This project is Tic tac toe game with redux

## See the project online

You can see the project online in [Tic tac toe](https://sirpak-ttt-game.surge.sh)

# Installation

```git
git clone https://github.com/mrsonmez/tic-tac-toe.git
cd tic-tac-toe
npm install
npm run dev
```
